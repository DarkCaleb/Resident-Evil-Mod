This Folder is for .con files.

Files tagged AlphaCode are for defining includes only.It should not have code in it.
Use it to include all your con modules with the right load order.

Files tagged BetaCode will always be first in your AlphaCode load order.
BetaCode files are for generic states and gamevars that can be used by multiple cons lower in the load order.



What is a con file:
In Duke 3D con files where used to attach code to tiles from a .art file.
it is used to make enemies,item and weapon pickups and breakable sprites.

In eduke32 con files can do almost anything.Too much too really talk about here.
